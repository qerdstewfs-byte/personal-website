# 页面内容交接文档

## 1. 使用了哪些插件或工具

- Shell / PowerShell：查看项目结构、读取页面文件、运行构建命令。
- apply_patch：创建和修改页面内容、组件与交接文档。
- Browser / Chrome：已用于本地页面展示和移动端检查。
- Node 静态服务器：用于临时服务 `dist` 构建产物，辅助 Browser 检查最终静态页面。
- Sites 插件：未使用。

- `/` 首页：已替换为个人学术技术主页内容。
- `/projects` 项目页：已完成项目卡片列表。
- `/research` 科研方向页：已完成六个研究方向说明。
- `/notes` 学习记录页：已完成分类式学习入口。
